# Image de fond pour les cartes cadeaux

## Instructions

Placez l'image de fond de la carte cadeau dans ce dossier avec le nom :
**`gift-card-background.jpg`**

### Spécifications de l'image :

- **Format** : JPG ou PNG
- **Dimensions recommandées** : 2100 x 1485 pixels (ratio A5 paysage)
- **Résolution** : 300 DPI pour une impression de qualité
- **Ratio** : 210:148.5 (format A5 paysage)

### Depuis l'image fournie :

L'image doit ressembler à celle que vous avez fournie, avec :
- Un arrière-plan avec des fleurs/roses
- Le texte "Un moment juste POUR toi" avec "BON CADEAU" et "NEWSAIIGE"
- Le slogan "NewSaiige c'est comme les chips, quand on commence on en devient ACCRO ! #NWSAIIGE"

### Zones de texte :

Les informations suivantes seront ajoutées automatiquement par le système sur le PDF :
- Pour : (nom du destinataire)
- Un Petit Mot d'❤️ : (message personnalisé)
- Bon pour un Soin : (montant en EUR)
- De la part de : (nom de l'acheteur)
- N° Bon Cadeau : (code unique)
- Valable jusqu'au : (date d'expiration)

Ces zones se trouvent dans la partie droite de la carte (coordonnées X: 102mm+).
